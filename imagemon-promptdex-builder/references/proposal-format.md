# Promptdex 提炼方案格式

写入新图鉴条目前，展示一份精简方案并等待用户一次确认。用户确认的是方案语义和范围，不是逐字措辞。

## 必需栏目

- 拟用模板名和用途描述。
- 生成任务或编辑任务。
- 最小输入集合及每项必需性。
- 将保留和通用化的规则摘要。
- 按原因分组的删除规则摘要。
- 所有拟加入的提炼补充、补充原因和不补充的影响；没有时明确写“无”。

删除摘要无需逐句罗列重复内容，但任何会显著改变原提示词视觉特征的删除必须单独指出。

## 确认有效性

确认后可以为清晰度、可执行性和去重调整措辞，但不得改变模板用途、输入集合及必需性、规则含义或已批准补充的含义。

任何影响模板名、用途、输入集合及必需性、保留规则、删除规则或提炼补充的修改都会使原确认失效，必须展示更新后的方案并重新等待确认。

## 精简示例

```md
模板名与用途：product-comparison-card，用于比较两个产品的关键差异
任务类型：生成任务
输入：products（必需）、audience（可选）
保留与通用化：双栏对比、清晰差异标记、有限强调色
删除：原产品名称和价格；平台专属参数；输出控制指令
提炼补充：无
```
